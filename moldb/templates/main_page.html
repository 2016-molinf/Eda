Popis:
Databáze primárně slouží k ukládání vybraných molekul. Obsahuje základní informace (jméno, SMILES, InCHI, InCHIKey, mol. hmotnost, sumární vzorec, 2D model molekuly), do budoucna se počítá s přidáváním informací o toxicitě, případně o biologické aktivitě látky. Vkládání látek je možno primárně z sdf souboru, případně ručně přes html formulář (omezená funkčnost).

Testovací odkazy:  Search the database,  Insert into the database,  vyhledávání,  zkouska pretty url funkce scitani,  ChemDoodle,  jsme.
